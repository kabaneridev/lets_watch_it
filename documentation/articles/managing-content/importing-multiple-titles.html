Importing Multiple Titles

Multiple titles can be imported quickly using themoviedb. There is no interface for this currently, however it can be done by entering import url directly in the browser: https://your-site.com/secure/tmdb/import?type=movie&page=1&limit=1000

Note:
This tool should only be used once after installation to quickly populate local database or occasionally afterwards. You should not try to use it via CRON or another automation tool.

Parameters

All parameters described here can be used, all are option, however you should be aware at least of these:

  • type: either movie or tv,Â this allows to import only movies or only tv shows.
  • page: number of page that should be imported from themoviedb.
  • limit: how many titles to import from a specific page.

Importing multiple pages

Multiple pages can be imported by changing page parameter. For example, to import five pages, you would open all these urls in the browser and wait until import is completed for each.

https://your-site.com/secure/tmdb/import?type=movie&page=1
https://your-site.com/secure/tmdb/import?type=movie&page=2
https://your-site.com/secure/tmdb/import?type=movie&page=3
https://your-site.com/secure/tmdb/import?type=movie&page=4
https://your-site.com/secure/tmdb/import?type=movie&page=5

Important:
Maximum page number is 1000, this is a limit on themoviedb side and there's no workaround for it currently.

Importing only specific genre

To import movies or series from specific genre only, you can use the url below. 24 and 48 can be replaced with any genre ID from themoviedb site.

https://your-site.com/secure/tmdb/import?type=movie&page=1&limit=100&with_genres=28,48

Importing based on keywords

Titles that have one or all keywords can be imported this way. 9717 can be replaced with any keyword ID from themoviedb site. Multiple keyword IDs can also be specified separated by a comma: 9717, 9718

https://your-site.com/secure/tmdb/import?type=tv&with_keywords=9717
https://your-site.com/secure/tmdb/import?type=movie&with_keywords=9717
